## 🔔 Fonctionnalité Phare : Notifications

### Comment ça marche ?

**Localisation** : En haut à droite → 🔔

**États** :
- Aucune notification : 🔔 (gris)
- 3 non lues : 🔔 **3** (rouge)

**Cliquer dessus** :

```
┌─────────────────────────────────────────┐
│ Notifications    [✓ Tout marquer]      │

│ ✅ Pack approuvé                       │

```

   ↓
