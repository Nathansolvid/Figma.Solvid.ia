# 🚀 Quick Start : Accès rapide à la transparence et aux notifications

## 🎯 TL;DR - En 30 secondes

### Pour voir la TRANSPARENCE d'un indicateur :
1. Menu latéral → **"Indicateurs clés"** 
2. Cliquez sur l'icône 👁️ **"Voir transparence"** sur n'importe quelle ligne
3. Explorez les 3 onglets : **Calcul**, **Historique**, **Validation**

### Pour voir vos NOTIFICATIONS :
1. Regardez en haut à droite → **Cloche 🔔**
2. Badge rouge = notifications non lues
3. Cliquez dessus pour voir le dropdown

## 🎯 Cas d'usage concrets

### Scénario 1 : Je veux vérifier un calcul

1. Menu → **"Indicateurs clés"**
2. Trouvez l'indicateur (ex: "Émissions GES totales")
3. Cliquez sur 👁️ **"Voir transparence"**
4. Onglet **"Calcul"** → Vérifiez la formule et les inputs
5. Onglet **"Historique"** → Voyez qui a modifié quoi
6. Cliquez sur **"Export PDF"** pour documenter

### Scénario 2 : Je veux suivre l'avancement d'un pack

1. Menu → **"Packs"**
2. Ouvrez un pack existant
3. Consultez l'onglet **"Checklist"** → Items à valider
4. Consultez l'onglet **"KPI Requirements"** → Indicateurs requis
5. Pour chaque KPI, cliquez sur 👁️ pour voir la transparence

### Scénario 3 : Un pack nécessite mon attention

1. Une notification 🔔 apparaît avec badge rouge
2. Cliquez sur la cloche
3. Lisez la notification (ex: "Pack prêt pour revue")
4. Cliquez sur la notification
5. **Vous êtes automatiquement redirigé vers le pack**

### Scénario 4 : Je suis auditeur et je veux valider des données

1. Menu → **"Audit Center"** (réservé aux auditeurs)
2. Voyez la file d'attente des packs à reviewer
3. Ouvrez un pack
4. Pour chaque indicateur, cliquez sur 👁️
5. Vérifiez les calculs et sources dans l'onglet **"Calcul"**
6. Consultez l'historique dans l'onglet **"Historique"**
7. Validez ou demandez des modifications

## 💡 Astuces

### Raccourcis clavier (fonctionnalités futures)
- `Ctrl+K` : Recherche globale
- `Ctrl+N` : Nouvelle notification (admin)
- `Escape` : Fermer modal

### Navigation rapide
- **Dashboard** → Vue d'ensemble
- **Packs** → Gestion par segments
- **Indicateurs clés** → Transparence complète
- **Audit Trail** → Historique global

### Performance
- Les notifications se rafraîchissent **automatiquement toutes les 30 secondes**
- Les données de transparence sont **cachées** pour éviter les rechargements inutiles
- Les packs se chargent avec **React Query** pour une expérience fluide

## 🎓 Prochaines étapes suggérées

1. **Créez votre premier pack** :
   - Menu → Packs → "Créer un nouveau pack"
   - Choisissez un template (ex: "Donneur d'Ordre")
   - Remplissez les informations

2. **Importez des données** :
   - Menu → Import données
   - Uploadez un fichier Excel/CSV
   - Mappez les colonnes
   - Validez l'import

3. **Ajoutez des preuves** :
   - Menu → Preuves & Documents
   - Uploadez des fichiers justificatifs
   - Liez-les aux indicateurs

4. **Exportez un livrable** :
   - Ouvrez un pack
   - Cliquez sur "Exporter PDF" ou "Exporter ZIP"
   - Téléchargez le dossier complet
